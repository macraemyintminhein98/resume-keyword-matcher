# Resume Keyword Matcher

**Quickly identify missing keywords in your resume based on a job description.**

This is a simple web application designed to help job seekers rapidly optimize their resumes for specific job postings. It takes a job description and your resume text, then highlights keywords from the job description that are absent from your resume.

## Features

*   **Job Description Input:** Paste or type in the job description.
*   **Resume Input:** Paste your resume text directly.
*   **Keyword Highlighting:** Instantly see keywords you're missing.
*   **Focus on Quantity:** Designed for speed, not complex NLP – get more applications out faster.

This project is built with Next.js/React. For local development:
